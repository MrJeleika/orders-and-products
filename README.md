# Description
System for control orders and products in shop or in stock


# Project setup

```
$ git clone https://github.com/MrJeleika/web-app-nest.git
```
### Install dependencies
yarn
```
$ yarn
```
npm
```
$ npm i
```
Create `.env.local` file with following variables:
```
API_HOST=Your api host
```
### Start development
yarn
```
$ yarn dev
```
npm
```
$ npm run dev
```
